The Redmi K20 Pro / Mi 9T Pro (codenamed _"raphael"_) is a flagship smartphone from Xiaomi announced in June 2019.

| Storage                 | 64GB / 128GB / 256GB                                            |

| Display                 | 1080 x 2340 pixels, 6.39 inches                                 |
